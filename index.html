Roland FP-30 Remote Control

The FP-30 Remote Control uses WebMIDI / Web Bluetooth to connect to an FP-30 instrument and control its dashboard settings. Not all dashboard settings have been implemented yet. It has been optimized for use with mobile browsers.

Both wired and BLE connections are supported. Note that connecting via BLE takes notably longer than MIDI (10 - 15 seconds are to be expected.) Once the connection is established the functionality is identical. If your browser doesn't support the BLE API no option for BLE will be shown.

It is self-contained, so you can save the control panel page locally and open it directly.

Compatible Browsers:

  • Google Chrome (except on iOS / iPadOS)
  • Chromium based browsers
  • Firefox (since 108, Desktop only, no Bluetooth)
Incompatible Browsers:
  • Safari (Apple refuses to implement WebMIDI in WebKit)
  • Google Chrome on iOS / iPadOS (Apple forces browsers to use WebKit)

The page uses the browsers local storage to store

  • selected MIDI device: (try to) reconnect on load.

This is considered experimental. It has only been tested with the FP-30. Instruments using the same mobile App (FP-10, FP-60) might (partially) work. Other instruments (especially newer instruments like the FP-30X) might work, not work, or end up doing weird things. Use at your own risk.
